# Système de Gestion de Restaurant

## Description
Application Java permettant la gestion des commandes, menus, tables,
facturation, stocks et rapports pour un restaurant.

## Technologies
- Java
- MySQL
- JDBC
- UML

## Installation
1. Cloner le dépôt
2. Importer la base via `database/init.sql`
3. Configurer `application.properties`
4. Lancer `Main.java`
